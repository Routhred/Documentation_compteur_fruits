pdf_maker.ecrireCell(xy: list, text: str, pdf: FPDF, width: Optional[int] = None)

Ecrire un text dans une cellule dans un pdf

Paramètres

  • xy (list) – position de la cellule

  • text (str) – text à écrire dans la cellule

  • pdf (fpdf.fpdf.FPDF) – pdr sur lequel ecrire

  • width (int) – taille de la cellule

Renvoie

None

Ecrire une entete dans un pdf

  • pdf (fpdf.fpdf.Fpdf) – pdf sur lequel on veut écrire

pdf_maker.ecrirePdf(data: list, output: str = 'Sortie', titre: str = 'Parcelles', date: Optional[str] = None)

Imprime en pdf les informations data

Avertissement

Chaque ligne du tableau doit avoir le même nombre de colonne

Note

Le pdf sera enregistré dans un dossier PDF dans le dossier parent

Paramètres

  • date (str) – date à ecrire dans l’entete si None, on met l’année actuelle

  • data (list) – tableau à 2 entrées que l’on veut imprimer

  • output (str) – nom de sortie du pdf

Renvoie

None
